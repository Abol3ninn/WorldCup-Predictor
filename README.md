# FIFA World Cup 2026 Predictor

A machine learning project that predicts the outcome of FIFA World Cup 2026 matches and estimates each team's probability of winning the tournament using machine learning and Monte Carlo simulation.

## Technologies

- Python
- Scikit-Learn
- XGBoost
- Pandas
- NumPy
- Matplotlib
- Football APIs
- Monte Carlo Simulation
